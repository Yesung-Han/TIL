인덱스에서 특정 id를 갖는 모든 도큐먼트들 삭제해달라는 요청이 들어와서 아래 쿼리 수행
=> term과 terms쿼리 조합으로 가능

```
POST 인덱스패턴명/_delete_by_query?pretty
{
  "query": {
    "bool": {
      "filter": [
        {
          "term": {
            "검색 필드1": "******"
          }
        },
        {
          "term": {
            "검색 필드2": "******"
          }
        },
        {
          "terms": {
            "검색 필드 3": [
              "1",
              "4",
              "6",
              "7",
              "8",
              "9",
              "10"
            ]
          }
        }
      ]
    }
  }
}

```
